modes

modedescription
words 30 common English words. The baseline — pure typing speed with no distractions.
time Countdown timer (15s, 30s, 60s, 120s, or custom up to 1 hour). Timer starts on first keypress.
quote A random literary excerpt. Punctuation, apostrophes, and sentence structure included.
code Real, idiomatic code snippets. Tab and Enter are live keystrokes. Syntax is highlighted as you type.

keyboard shortcuts

ctrl+rrestart with new text tabtype a tab character (code mode) entertype a newline (code mode) escstart a new test

why this exists

Most typing tests are designed for general users — random words, no code, accounts required for history. Typist is built for developers who want to practice the patterns they actually type every day: function signatures, conditionals, closures, match arms. It runs offline, stores scores locally, and ships as a single binary with a built-in web UI. No telemetry. No ads. No paywall.

terminal ui

This web version and the terminal UI share the same codebase and score storage. The TUI adds blind mode, session history, JSON/CSV export, theme switching (Mocha ↔ Latte), and focus mode. Install it with: 

go install github.com/chuma-beep/typist@latest
